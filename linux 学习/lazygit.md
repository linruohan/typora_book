## lazygit下载（delta）和配置：

```bash
#下载lazygit https://github.com/jesseduffield/lazygit
lazygit_0.31.4_Linux_x86_64.tar.gz
#下载delta：https://github.com/dandavison/delta#installation
delta-0.10.2-x86_64-unknown-linux-gnu.tar.gz 
```
