#### 更多三个点

```bash
Copy to clipboard
Dumplicate
Move                  移到其他的project
Repeat               重复设置：Daily Weekly Monthly Yearly -- None Custom（Every）

Delete task
-----
Added at: Today 20:36
Updated at: Today 21:37
```

重复设置

``` 
Repeat
Every 2 days
2 - + [Day(s)) Week(s) Month(s) Year(s)]

Week(s): Monday Tuesday Wednesday Thusday Friday Saturday Sunday
Done 按钮
```
